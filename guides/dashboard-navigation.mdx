## Main Dashboard Elements

### Statistics Cards

The dashboard displays key financial metrics:

<AccordionGroup>
  <Accordion title="Transaction Statistics">
    - **Total Transactions**: Count of all transactions
    - **Total Amount**: Sum of transaction amounts
    - **Categories**: Number of user categories
    - **CSV Uploads**: Number of imported files
  </Accordion>
  
  <Accordion title="Bill Patterns">
    - **Active Patterns**: Currently tracked recurring bills
    - **This Month Total**: Total recurring expenses
    - **Overdue**: Bills that are overdue
    - **Upcoming**: Bills due soon
  </Accordion>
  
  <Accordion title="Expense Tracking">
    - **One-time Expenses**: Non-recurring expenses
    - **Income**: Monthly income tracking
    - **Net Total**: Income minus expenses
    - **Today's Expenses**: Current day spending
  </Accordion>
</AccordionGroup>

## Keyboard Shortcuts

<Info>
Use keyboard shortcuts for faster navigation and improved productivity.
</Info>

### Common Shortcuts

- **Ctrl + D**: Go to Dashboard
- **Ctrl + T**: Go to All Transactions
- **Ctrl + E**: Create Expense
- **Ctrl + B**: Create Bill
- **Ctrl + U**: Upload CSV
- **Ctrl + S**: Save changes
- **Esc**: Close dialogs
