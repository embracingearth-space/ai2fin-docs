## Accessing Transaction Management

### From Dashboard

1. **Click "All TXns"** in the sidebar
2. **View Overview** of all transactions
3. **Use Quick Actions** for common tasks

### From Navigation

1. **Sidebar Menu** → "All TXns"
2. **Direct URL**: `/all-transactions`
3. **Keyboard Shortcut**: Ctrl + T

## Transaction Operations

### Individual Transaction Actions

<Steps>
  <Step title="Edit Transaction">
    1. Click the edit icon on any transaction
    2. Modify description, amount, category
    3. Update tax deductibility
    4. Save changes
  </Step>
  
  <Step title="AI Categorize">
    1. Click the AI categorize button
    2. Review AI suggestions
    3. Accept or modify category
    4. Apply changes
  </Step>
  
  <Step title="Delete Transaction">
    1. Click the delete icon
    2. Confirm deletion
    3. Transaction is soft-deleted
  </Step>
</Steps>

Process multiple transactions at once:

<AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Slow Loading">
    **Causes:**
    - Large number of transactions
    - Complex filters
    - Network issues
    
    **Solutions:**
    - Use date filters
    - Reduce page size
    - Check internet connection
    - Clear browser cache
  </Accordion>
  
  <Accordion title="Missing Transactions">
    **Causes:**
    - Date filters applied
    - Category filters
    - Search filters
    
    **Solutions:**
    - Clear all filters
    - Check date range
    - Verify search terms
    - Refresh the page
  </Accordion>
  
  <Accordion title="AI Not Working">
    **Causes:**
    - OpenAI API key not set
    - Service temporarily down
    - Network connectivity
    
    **Solutions:**
    - Check API configuration
    - Verify service status
    - Try again later
    - Contact support
  </Accordion>
</AccordionGroup>
