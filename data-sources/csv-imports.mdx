# CSV Imports

AI2Fin's CSV Import feature allows you to easily import bank statements, transaction files, and financial data from various sources. This is the primary way to get your financial data into the platform.

## Accessing CSV Import

### From Dashboard

1. **Navigate to Dashboard**: Click "Dashboard" in the sidebar
2. **Click "Upload CSV"**: Use the quick action button
3. **Select File**: Choose your CSV file
4. **Configure Import**: Map fields and set options

### From Bank Transactions

1. **Go to Bank Transactions**: Click "Bank Transactions" in the sidebar
2. **Click "Upload CSV"**: Use the upload button
3. **Follow Import Wizard**: Complete the import process

### From All Transactions

1. **Navigate to All Transactions**: Click "All TXns" in the sidebar
2. **Click "Import Data"**: Use the import option
3. **Select CSV Import**: Choose CSV as import type

## Import Process

<Steps>
  <Step title="Prepare Your File">
    **File Requirements:**
    - Format: CSV (Comma Separated Values)
    - Encoding: UTF-8 recommended
    - Size: Up to 10MB per file
    - Columns: Date, Description, Amount (minimum)
    
    **Common Column Names:**
    - Date: "Date", "Transaction Date", "Posted Date"
    - Description: "Description", "Memo", "Payee"
    - Amount: "Amount", "Debit", "Credit"
    - Balance: "Balance", "Running Balance"
  </Step>
  
  <Step title="Upload File">
    1. **Click "Choose File"**: Select your CSV file
    2. **Verify File**: Check file name and size
    3. **Click "Upload"**: Start the upload process
    4. **Wait for Processing**: File is uploaded and parsed
  </Step>
  
  <Step title="Map Fields">
    1. **Review Columns**: See detected column headers
    2. **Map Fields**: Match CSV columns to AI2Fin fields:
       - **Date**: Transaction date
       - **Description**: Transaction description
       - **Amount**: Transaction amount
       - **Balance**: Account balance
       - **Reference**: Transaction reference number
    3. **Set Options**: Configure import settings
    4. **Preview Data**: Review first few rows
  </Step>
  
  <Step title="Configure Settings">
    **Import Options:**
    - **Date Format**: Specify date format (DD/MM/YYYY, MM/DD/YYYY, etc.)
    - **Currency**: Set currency for transactions
    - **Account**: Assign to specific account
    - **Skip Duplicates**: Avoid importing duplicate transactions
    - **Auto-Categorize**: Use AI to categorize transactions
  </Step>
  
  <Step title="Complete Import">
    1. **Review Preview**: Check imported data
    2. **Confirm Import**: Click "Import Transactions"
    3. **Monitor Progress**: Watch import status
    4. **Review Results**: Check imported transactions
  </Step>
</Steps>

### Duplicate Handling

<Warning>
Duplicate transactions can cause data inconsistencies and reporting errors.
</Warning>

**Duplicate Detection Methods:**
- **Exact Match**: Same date, amount, and description
- **Fuzzy Match**: Similar transactions within tolerance
- **Reference Match**: Same reference number
- **Manual Review**: Flag potential duplicates for review

## API Integration

For developers, CSV import is available via API:

```javascript
// Upload CSV file
const formData = new FormData();
formData.append('file', csvFile);
formData.append('accountId', 'acc_123');
formData.append('currency', 'AUD');

const response = await fetch('/api/import/csv', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer ' + token },
  body: formData
});

const result = await response.json();
// Returns: { importId: 'imp_123', status: 'processing' }
```
