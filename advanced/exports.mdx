# Tax Export

Complete guide to generating ATO-compliant tax exports and reports from AI2Fin for seamless tax return preparation and compliance.

## Overview

AI2Fin's Tax Export feature generates comprehensive, ATO-compliant reports that streamline your tax return preparation. Export transaction data, travel expenses, and deductions in formats ready for submission to the ATO.

### From Dashboard

## ATO Compliance Features

### Required Fields

<Info>
All exports include ATO-required fields for tax compliance.
</Info>

**Mandatory Fields:**
- **Transaction Date**: Date of transaction
- **Description**: Transaction description
- **Amount**: Transaction amount
- **Category**: Expense category
- **Tax Deductible**: Yes/No status
- **Business Use %**: Percentage for business use
- **Deduction Amount**: Calculated deduction
